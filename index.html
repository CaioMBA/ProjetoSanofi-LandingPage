  • Micro WMS

    O projeto tem como objetivo criar um sistema de gerenciamento de estoque e pedidos, com o intuito de facilitar a gestão de produtos e pedidos, tornando o processo mais eficiente e rápido.

    GitHub

Objetivos

  • Cadastros Gerais

    Será possível cadastrar produtos tanto físicos quanto digitais, facilitando assim a dinâmica e a flexibilidade do sistema.

  • Controle de Estoque

    Rastreabilidade e controle de produtos cadastrados e em estoque, para que seja possível atender as necessidades com maior velocidade

  • Chamados

    Será possível realizar pedidos e chamados para que a equipe responsável possa atender as demandas com maior eficiência
